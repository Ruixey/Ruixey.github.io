Coding Conventions

Finding A Name​

  • Should contain no Shortcuts
  • Should be as descriptive as possible

Classes​

  • First letter always uppercase
  • CamelCase – writing style (f.e. CoffeeMachine)
  • Use only nouns (f.e. Coffee)
  • Correct spelling

Functions​

  • First letter always lowercase
  • CamelCase – writing style (f.e. makeCoffee)
  • use verbs + nouns (f.e. makeCoffee)
  • Correct spelling

Variables​

  • First letter always lowercase
  • CamelCase – writing style (f.e. coffeeReceipe)
